Welcome to Release Coordination I-Signon.

Purpose of I-Signon

The purpose of this site is to support the efforts of the Production and Release Coordination Services (PARCS) team in managing, and tracking progress of signon activities.

Profile Registration

If your area has been invited to participate in a release family, please complete a release family profile by selecting the "Register Profile" menu selection above.

Signon Confirmation

If you are confirming your team's participation in a particular signon, please complete the required confirmation form by selecting "Confirm Signon Participation" menu selection above.
